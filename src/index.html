🐳 Bienvenido a la Página Web

Página desplegada con Dockerhub y Nginx

Bienvenido

Este es un sitio web estático creado con Nginx y a su vez lanzado mediante un contenedor en Docker.

Si usted puede ver las instrucciones la página se lanzo sin fallas.

¿Qué se uso?

  • 🐳 Docker
  • ⚙️ Nginx (Alphine)
  • 📄 Html y estilos
  • ☁️ DockerHub

Información del Proyecto

Este proyecto forma parte de la primer unidad en Programación Web donde los puntos principales son:

  • Crear un sitio web estático con Html
  • Crear un Dockerfile funcional
  • Contruir y ejecutar una imagen en Docker
  • Publicar Dicha Imagen en DocerHub
  • Colocar un .Dockerignore para archivos innecesarios
  • Crear un Readme.md claro para poder usarlo
  • Usar GitHub como medio de documentación del proyecto.
